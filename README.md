# Chili Catcher

Chili Catcher is a Foundry VTT minigame module where players catch falling chili peppers.

## Features

- Lightweight reaction minigame inside Foundry VTT
- GM aware score handling with deduplication safeguards
- Local audio for start and game over moments
- English and German localization
- Handlebars based UI

## Installation

1. Open **Add-on Modules** in Foundry VTT.
2. Click **Install Module**.
3. Paste this manifest URL:
   `https://github.com/LisaNeoGermany/chili-catcher/releases/latest/download/module.json`
4. Click **Install**.

## Compatibility

- Minimum Foundry version: 13
- Verified Foundry version: 14
